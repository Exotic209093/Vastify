## How the video itself was built

The video was assembled with **[Hyperframes](https://hyperframes.heygen.com)** — HeyGen's open-source HTML-based video framework. Each scene is an `index.html` composition with a paused GSAP timeline that the framework scrubs frame-by-frame in headless Chromium and pipes to FFmpeg.

Local generation pipeline (regenerable at any time from the source HTML):

1. Each scene is a Hyperframes project (`v2/intro/`, `v2/setup/`, `v2/diff/`, `v2/rules/`, `v2/architecture/`, `v2/dashboard/`, `v2/savings/`, `v2/outro/`, `v2/why-claude/`)
2. Voice generated by **ElevenLabs** API (one MP3 per scene, see `generate-elevenlabs.mjs`)
3. Music synthesised in FFmpeg (layered sines + lowpass + tremolo + soft echo)
4. Per-scene audio padded to scene duration, concatenated into a single `narration-eleven-mixed.wav`
5. Video stitched via FFmpeg `concat` filter, muxed with the audio track
6. Final speed-up: `setpts=PTS/1.10` on video + `atempo=1.10` on audio to fit the 3-minute cap

The build artefacts (`v2/final/vastify-3min-eleven-final.mp4`, individual scene MP4s, MP3s, WAVs) are **gitignored** because they're large (~200 MB total) and trivially regenerable. Source HTML and narration text files would normally live here too, but were also gitignored to keep the repo lean — they're available on request or in the source machine's working tree.
